# 命令

1. gitbook init # 根据summary.md生成对应的目录和文件
2. gitbook serve # 根据md文件生成对应的静态文件，可以在本地端口查看
3. gitbook pdf ./图书目录　＃pdf文件
4. 安装插件后。
   1. gitbook build

   2. gitbook serve

   3. ```latex

      ```
